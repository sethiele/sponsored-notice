Add a "Sponsored" notice to your posts.

== Description ==
You need to add a "sponsored" notice to your posts? This plugin lets add this notice to mosts and every place you need it.

== Installation ==
1. Upload the `sponsored-notice` folder to the `/wp-content/plugins/` directory
1. Activate the *Add sponsored notice* plugin through the \'Plugins\' menu in WordPress
1. Configure the plugin by going to the `Sponsored` menu that appears in your admin menu
